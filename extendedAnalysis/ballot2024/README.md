# Ballot 2024

Their works describes a way to convert augmented attack graphs
into (augmented) timed automata, timed priced Markov decision processes 
to be exact.
To set it all up, simply run the provided setup.sh.
The functionalities provided in derivTree2xml.py translates 
the annotated derivation tree extracted from the ProVerif model
into the input format expected by Ballot's tool.
